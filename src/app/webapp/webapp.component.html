PWA

Eine Progressive Web Application erhält das Aussehen und die Funktionalität einer nativen Anwendung. Damit geht die parallele Entwicklung von Websites und Apps endlich zu Ende. Diese Applikation lässt sich wie vertraut installieren, inklusive Icon auf dem Home Bildschirm.

Offline

Ein Service Worker garantiert eine einwandfreie Funktionalität auch offline. Durch eine ausgeklügelte Optimierung ist die Ladezeit bei jeder Verbindung erheblich verkürzt.

Standard

Mit der stetig wachsenden Unterstützung der modernen Web Technologien erweitern sich fast täglich die Einsatzmöglichkeiten dieser offenen Standards. Fetch API, Push API, Cache API, IndexedDB, Gamepad API, Web Speech.
